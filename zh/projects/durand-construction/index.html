Durand Construction

为位于布拉柴维尔的 Durand Construction 开发的企业展示网站，该公司专注于建筑和设计领域。网站通过清晰且结构化的布局展示其项目、服务和技术方法。

Astrojs Preactjs Css

概述

Durand Construction 位于 布拉柴维尔 ，专注于建筑施工和工程领域。 本展示网站旨在以简洁直观的方式呈现公司的项目、服务及工作理念。

设计

界面采用__模块化布局__ ，核心基于简洁排版和建筑设计启发的视觉节奏。 页面结构突出关键板块——

作品、关于和博客——呈现清晰且平衡的层次。 网站采用 __静态架构__开发，优先保证速度、安全性和低维护成本。

内容

网站展示了近期精选项目及提供的服务，并包含一个专门的__博客__ 区域， 分享有关建筑、可持续发展及行业教育的文章。 总体目标是增强公司的线上影响力，并通过简洁精致的方式展示专业能力。

技术栈

项目依托多项互补技术：Astro 用于静态网站生成，保证速度与性能； TypeScript 提供可靠且易维护的代码；CSS 负责视觉结构与样式； Markdown 实现简单高效的内容管理。
