基于 Tiptap 的 Markdown 编辑器

一款基于 Web 的 Markdown 编辑器，提供流畅现代的写作体验，支持实时预览、自定义工具栏和内容导出功能。采用 Tiptap、React、TypeScript 和 SCSS 构建。

编辑器设计

本项目旨在构建一款现代 Markdown 编辑器，将纯文本的简洁性与所见即所得界面的强大功能相结合。选择 Tiptap 是因其灵活性及与 React 的无缝集成。界面设计 直观且简约，为技术写作者和开发者提供流畅愉悦的写作体验。

自定义工具栏与 Markdown 快捷键

编辑器包含自定义工具栏 ，用户可轻松应用加粗、斜体、标题、列表和引用等格式。同时支持原生 Markdown 快捷键，实现快速自然的写作。所有 UI 组件使用 SCSS 样式，确保视觉一致性和跨屏幕尺寸的响应式表现。
